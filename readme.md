1) Clone Repo
~~~
git clone -b develop https://github.com/Opentribes/Core.git /path/to/your/folder
~~~
2) Install dependencies
~~~
cd /path/to/your/folder
composer update -o 
~~~

3) Test features
~~~
bin/behat
~~~
